# 项目规则与用户偏好

本文件用于记录用户的偏好、自定义规则和项目特定的配置。

## 项目信息

- **项目名称**: 森源管理系统
- **技术栈**: Next.js + TypeScript + Electron
- **包管理器**: pnpm

---

## 用户偏好

### 通用偏好
- 语言: 中文
- 包管理器: pnpm (不使用 npm 或 yarn)

### 代码风格
- TypeScript 严格模式
- React Hooks 方式开发组件

---

## 项目特定规则

### PLC 相关功能
- 支持32位浮点寄存器读写
- 支持线圈M读写
- 不使用64位浮点寄存器功能（已删除）

### 文件结构
- API路由位于 `/src/app/api/`
- 页面组件位于 `/src/app/`
- UI组件位于 `/src/components/`
- 工具函数和服务位于 `/src/lib/`
